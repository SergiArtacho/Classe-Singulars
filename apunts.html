L'estructura d'HTML

En el moment de crear el document base, l'estructura d'HTML és la següent:

    <!DOCTYPE html>
    <html lang="en">
    <head>
        <meta charset="UTF-8">
        <title>Document</title>
        <style></style>
    </head>
    <body>
        
    </body>
    </html>

El text comença indicant-nos el format <!DOCTYPE html> , seguidament hem d'indicar la llengua en que escriurem la nostre web , <html lang="en">.

L'element <head> és un contenidor per a metadades (dades sobre dades) i se situa entre l'etiqueta html i l'etiqueta <body>.Les metadades HTML són dades sobre el document HTML. Les metadades no es mostren.Normalment, les metadades defineixen el títol del document, el conjunt de caràcters, els estils, els scripts i altra informació meta. L'element <body> conté tot el contingut d'un document HTML, com ara encapçalaments, paràgrafs, imatges, hiperenllaços, taules, llistes, etc. Nota: Només hi pot haver un element <body> en un document HTML. Per configurar l'estètica en aquest cas amb el CSS, L’atribut <style> s’utilitza per afegir estils a un element, com ara color, tipus de lletra, mida i molt més.

Si volem fer comentaris sobre el codi sense que s'executin, a Html, hem de fer <!----> i a CSS /* Comentari */

Etiquetes

p és una etiqueta (tag), style és un atribut (atribute), com menys etiquetas millor, la majoria de coses ho pots fer a style directament. És a dir, encomptes d'escriure el codi a cada línea, ho configurem previament a style.

  • Pre–––>Tot el que hi hagi dintre d'un pre s'interprenta (els espais incolosos). Útil per posar trossets de codi. 

    Exemple de pre

  • p––––> Per fer paràgrafs

  • span––––> Per canviar l'estil de unes poques línies

  • hr––––> Per fer una línia horitzontal

  • img––––> Per insertar imatges, és pot fer tant amb imatges guardades al pc(millor) o amb url. Src per marcar la ruta o enllaç i alt per posar el text que apareix al posar el cursor un cop a la web.

  • head–––> indica que comença la capçalera de la pàgina. S'hi sol posar el títol (<title>) de la web, una descripció i altres informacions relacionades amb el contingut de la pàgina.

  • body––––> Es el cos de la pàgina, on va el que es veu en el navegador al carregar la pàgina. En el body van els textos, les imatges i tots els continguts del web.

Estil

Dintre d'estil, podem configurar l'estetica, com per exemple en el text, com el color, el fons del text... La funció margin:auto, indica els marges de manera automatica.

Hello world

  codi: <p style="margin-block-start: 2em; color: aqua;"> Hello world</p>

Hola mundo en castellà

  codi: <p style="background-color: blanchedalmond;">Hola mundo en castellà</p>

background-color: Estableix el color de fons d'un element. El fons d'un element és la mida total de l'element, inclòs el farciment i la vora (però no el marge). És pot fer també amb imatges, background-image.

Llista

Trobem dos tiupus de llista:

Llista desordenada

  • Laia
  • Julian
  • Óscar

Per fer una llista desordenada utilitzem <ol> i per cada element de la llista, ho fem amb <li>

Llista ordenada

  1. Laia

  2. Julian

  3. Óscar

Per fer una llista ordenada utilitzem <ol> i per cada element de la llista, ho fem amb <li>.

A la llista ordenada hi apareixen números.

Estil dels elements dintre de les llistes

  1. Laia

  2. Julian

  3. Óscar

Si volem afegir estil dintre de la llista, ho podem fer afegint una classe (el més recomanable) o afegint l'estil a cada línea, com hem fet en l'anterior llista. Codi llista: 

 
    <ol> 
        <li><p style="margin: 0; font-style: inherit;">Laia</li>
        <li> <p style="margin: 0; font-style: oblique;"> Julian </p></li>
        <li><p style="text-decoration: underline; margin: 0;"> Óscar</li>
    </ol>

Sobre CSS

El CSS (fulls d'estil en cascada) és un llenguatge que defineix l'aparença d'un document escrit en un llenguatge de marcat (per exemple, HTML). ... Aquesta separació entre l'estructura i la presentació és molt important, ja que permet que només canviant els CSS es modifiqui completament l'aspecte de la nostra pàgina.

Pots fer marge directament, o seleccionar entre els 4 marges. O sobre la mateixa funció original a través de 0 0 0 0.

Display

La propietat de visualització especifica el comportament de visualització (el tipus de quadre de representació) d'un element. En HTML, el valor de propietat de visualització per defecte es pren de les especificacions HTML o del full d'estil predeterminat del navegador / usuari. El valor per defecte a XML és en línia, inclosos els elements SVG.

  • Inline––––> Mostra un element com a element en línia. Les propietats d’alçada i amplada no tindran cap efecte

  • Block––––> Mostra un element com a element de bloc. Comença per una línia nova i ocupa tota l'amplada

  • Flex–––> Mostra un element com a contenidor flex a nivell de bloc

  • Gird––––> Mostra un element com a contenidor de quadrícula a nivell de bloc.

  • Inline-block––––> Mostra un element com a contenidor de blocs a nivell de línia. L'element en sí té el format d'un element en línia, però podeu aplicar valors d'alçada i amplada

  • Inline-flex––––––> Mostra un element com a contenidor flexible a nivell de línia.

  • Inline-gird––––––> Mostra un element com a contenidor de quadrícula en línia

  • Display-none: Es pot ocultar un element configurant la propietat de visualització a cap. L'element quedarà amagat i la pàgina es mostrarà com si l'element no hi fos

Float

La propietat CSS float situa un element a la banda esquerra o dreta del seu contenidor, permetent als elements de text i en línia aparèixer al seu costat. L'element és remogut del flux normal de la pàgina, tot i que encara segueix sent part d'ell (a diferència de l'posicionament absolut).

Hover

El selector: hover s'utilitza per seleccionar elements quan es passa el ratolí sobre ells. Es pot utilitzar en tots els elements, no només en els enllaços.

coincideix quan l'usuari interactua amb un element amb un dispositiu apuntador, però no necessàriament ho activa. Generalment s'activa quan l'usuari es desplaça sobre un element amb el cursor (punter de ratolí).

Exemple:

  • Això és un hover

CODI:


        HTML:
 <ul>
        <li class="hoverex"><a href="#" >
                Això és un hover</a>
        </li>

    </ul>


        CSS:

        hoverex {
            background-color:black;
            border: 1px solid;
            list-style: none;
            max-width: max-content;
        }



        .hoverex a{
            text-decoration: none;
            color: yellow;
        }

        .hoverex:hover{
            background-color: yellow;
            transform: scale(1.2);
        }

        .hoverex a:hover{
            color: black;
        }

Git Hub

GIT: Git és un sistema de control de versions de codi obert. Els desenvolupadors apliquen canvis en els seus codis tot el temps. Primer per a les versions beta, després vénen les versions estables o definitives, i també després de l'llançament de l'aplicació o lloc web que estan alternant.

GitHub és un aplicació web per a control de canvis i documentació, que funciona alhora com hosting de codi i xarxa social entre desenvolupadors.

Funcions Git Hub:

  • Funciona a mode de repositori

  • Control de canvis

  • Documentació de projectes

  • És un sistema segur

  • És pot utilitzar directament des de Visual Studio Code

Comandos esencials de Git:

  • Git init–––> Crea el repositori Git en el local per que puguis començar a treballar.

  • Git add–––> Utilitza git add per afegir arxius a l'índex de el projecte. Aquest comando s'utilitza afegint a la fin el nom de l'arxiu amb un espai a la fi de la comanda com ho veus en l'exemple sota.

  • Git commit––––> Guarda els canvis realitzats amb git commit. Al desar podràs especificar un missatge per identificar el canvi que vas realitzar o afegir un comentari. Exemple: git commit -m "mensaje de referencia"

  • Git branch––––> Aquest comando es fa servir per llistar, crear o esborrar branques. Per llistar totes les branques utilitza:

  • Git stash––––> S'utilitza per guardar canvis temporals

  • Git log––––> Pots veure una llista dels commits realitzats.

  • Git tag–––> S'utilitza per etiquetar commits especifics, per exemple: git tag 1.0.1 <id de commit>

  • Git diff––––> Llista de conflictes existents

  • Git merge–––> per fusionar un branch actiu amb un altre.
